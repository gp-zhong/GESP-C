C++ 基本数据类型

掌握 C++ 中的基本数据类型，它们的特性、范围和用法

数据类型概览

C++ 提供了丰富的数据类型，用于存储不同种类的数据。这些数据类型主要分为以下几类：

基本数据类型

  • 整型（整数）
  • 浮点型（小数）
  • 字符型
  • 布尔型

复合数据类型

  • 数组
  • 字符串
  • 结构体
  • 类（class）

其他数据类型

  • 指针
  • 引用
  • 枚举
  • 空类型（void）

在本文中，我们将重点介绍 C++ 基本数据类型的特性、范围和用法，包括整型、浮点型、字符型和布尔型。

整数类型（Integer Types）

整数类型用于存储整数值（没有小数部分的数字）。C++ 提供了多种整数类型，它们的区别主要在于占用的内存空间和可表示的数值范围。

基本整数类型

类型 大小 值范围 用途
short int 2 字节 -32,768 到 32,767 小范围整数
int 4 字节 -2,147,483,648 到 2,147,483,647 常用整数类型
long int 4 字节 同 int 中等范围整数
long long int 8 字节 -9,223,372,036,854,775,808 到 9,223,372,036,854,775,807 大范围整数

注意：

上述大小是针对大多数现代计算机系统。在不同操作系统和编译器下，这些类型的大小可能略有不同。可以使用 sizeof 运算符来获取当前环境下的准确大小。

无符号整数类型

在整数类型前加上 unsigned 关键字，可以将其转换为无符号类型，它只能存储非负整数，但是可以表示的正整数范围会更大。

类型 大小 值范围
unsigned short int 2 字节 0 到 65,535
unsigned int 4 字节 0 到 4,294,967,295
unsigned long int 4 字节 0 到 4,294,967,295
unsigned long long int 8 字节 0 到 18,446,744,073,709,551,615

整数类型的使用示例

#include <iostream>
using namespace std;

int main() {
    // 整数类型声明与初始化
    int a = 10;
    short b = 20;
    long c = 30L;          // L 后缀表示 long 类型
    long long d = 40LL;    // LL 后缀表示 long long 类型
    
    // 无符号类型
    unsigned int e = 50U;  // U 后缀表示 unsigned
    
    // 输出各变量值和大小
    cout << "int a = " << a << ", 大小: " << sizeof(a) << " 字节" << endl;
    cout << "short b = " << b << ", 大小: " << sizeof(b) << " 字节" << endl;
    cout << "long c = " << c << ", 大小: " << sizeof(c) << " 字节" << endl;
    cout << "long long d = " << d << ", 大小: " << sizeof(d) << " 字节" << endl;
    cout << "unsigned int e = " << e << ", 大小: " << sizeof(e) << " 字节" << endl;
    
    // 整数类型的范围
    int max_int = 2147483647;
    int min_int = -2147483648;
    cout << "int 最大值: " << max_int << endl;
    cout << "int 最小值: " << min_int << endl;
    
    // 整数溢出示例
    int overflow = max_int + 1;
    cout << "溢出后的值: " << overflow << endl;  // 将输出 -2147483648（回绕到最小值）
    
    return 0;
}

整数类型的选择

  • 一般情况下，int 是最常用的整数类型，适用于大多数场景
  • 当需要节省内存且数值范围较小时，可以选择 short
  • 对于较大的整数，使用 long long
  • 如果确定数值不会为负，可以使用 unsigned 类型来扩大正数范围
  • 在算法竞赛中，经常使用 long long 来避免整数溢出问题

整数溢出：当计算结果超出数据类型的表示范围时，会发生整数溢出。在 C++ 中，有符号整数的溢出是未定义行为，而无符号整数的溢出会按模运算处理（即回绕）。在编写程序时应当注意避免整数溢出，特别是在处理大数据和执行算术运算时。

浮点类型（Floating-Point Types）

浮点类型用于表示包含小数部分的数值。C++ 提供了三种浮点类型，它们的区别在于精度和范围。

基本浮点类型

类型 大小 精度 用途
float 4 字节 7 位有效数字 低精度浮点数
double 8 字节 15-16 位有效数字 标准浮点数，常用
long double 至少 8 字节，通常 12 或 16 字节 至少 15 位有效数字，通常更多 高精度浮点数

浮点数表示

浮点数在计算机中以科学计数法的形式表示：一个有符号的尾数和一个有符号的指数。例如，数字 123.45 可以表示为 1.2345 × 10²。

由于这种表示方式，浮点运算可能会有精度损失，特别是在执行大量计算或比较浮点数时。

浮点类型的使用示例

#include <iostream>
#include <iomanip>  // 用于输出格式控制
using namespace std;

int main() {
    // 浮点类型声明与初始化
    float a = 3.14f;       // f 后缀表示 float 类型
    double b = 3.14159;    // 默认浮点常量为 double 类型
    long double c = 3.141592653589793238L;  // L 后缀表示 long double 类型
    
    // 设置输出精度
    cout << fixed << setprecision(10);
    
    // 输出各变量值和大小
    cout << "float a = " << a << ", 大小: " << sizeof(a) << " 字节" << endl;
    cout << "double b = " << b << ", 大小: " << sizeof(b) << " 字节" << endl;
    cout << "long double c = " << c << ", 大小: " << sizeof(c) << " 字节" << endl;
    
    // 科学计数法表示
    double d = 1.23e5;  // 1.23 × 10^5 = 123000
    double e = 4.56e-3; // 4.56 × 10^-3 = 0.00456
    cout << "科学计数法 d = " << d << endl;
    cout << "科学计数法 e = " << e << endl;
    
    // 浮点数精度问题演示
    double f = 0.1 + 0.2;
    cout << "0.1 + 0.2 = " << f << endl;  // 可能不等于 0.3，而是非常接近 0.3 的数
    
    // 浮点数比较
    if (f == 0.3) {
        cout << "f 等于 0.3" << endl;
    } else {
        cout << "f 不等于 0.3（浮点精度问题）" << endl;
        cout << "正确的比较方式是使用一个很小的误差范围" << endl;
        
        double epsilon = 1e-10;  // 一个很小的误差范围
        if (abs(f - 0.3) < epsilon) {
            cout << "f 约等于 0.3" << endl;
        }
    }
    
    return 0;
}

浮点类型的选择

  • double 是最常用的浮点类型，精度足够大多数应用
  • 当存储空间受限且不需要高精度时，可以使用 float
  • 当需要极高精度的计算时，可以考虑 long double
  • 在算法竞赛中，通常使用 double 进行浮点计算

浮点数的注意事项

  • 精度限制：浮点数只能表示有限位数的有效数字，超出部分会被截断或舍入
  • 不精确的二进制表示：许多十进制小数（如 0.1, 0.2）在二进制中无法精确表示
  • 比较问题：不应直接使用 == 比较两个浮点数，而应使用误差范围
  • 累积误差：在大量浮点运算中，微小的误差可能会累积为较大的偏差
