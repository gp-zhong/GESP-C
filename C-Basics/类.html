C++ 类

掌握 C++ 面向对象编程的核心概念，理解类的定义与应用

类的基础

类是 C++ 面向对象编程的基础，它是用户自定义的数据类型，将数据和操作数据的函数封装在一起。

// 类的基本语法
class 类名 {
    // 访问修饰符: private, protected, public
private:
    // 私有成员（默认）
    
protected:
    // 受保护成员
    
public:
    // 公有成员
    
    // 构造函数
    类名(参数列表);
    
    // 析构函数
    ~类名();
    
    // 成员函数
    返回类型 函数名(参数列表);
};

// 示例：定义一个简单的类
class Rectangle {
private:
    double length;
    double width;
    
public:
    // 构造函数
    Rectangle(double l = 0, double w = 0) : length(l), width(w) {}
    
    // 成员函数
    double getArea() {
        return length * width;
    }
    
    double getPerimeter() {
        return 2 * (length + width);
    }
    
    // 访问器和修改器
    double getLength() { return length; }
    void setLength(double l) { length = l; }
    
    double getWidth() { return width; }
    void setWidth(double w) { width = w; }
};

类的组成部分：

  • 数据成员：存储类的状态的变量
  • 成员函数：操作类数据的函数
  • 访问修饰符：控制对类成员的访问权限
  • 构造函数：初始化类对象的特殊函数
  • 析构函数：在对象被销毁时自动调用的函数

对象创建与使用

类定义了对象的结构，一个类可以创建多个对象实例，每个实例有自己的数据成员。

// 创建和使用对象
int main() {
    // 创建对象
    Rectangle rect1(5.0, 3.0);
    Rectangle rect2;  // 使用默认参数
    
    // 使用对象的成员函数
    cout << "矩形1的面积: " << rect1.getArea() << endl;
    cout << "矩形1的周长: " << rect1.getPerimeter() << endl;
    
    // 通过修改器设置数据
    rect2.setLength(4.0);
    rect2.setWidth(2.5);
    cout << "矩形2的面积: " << rect2.getArea() << endl;
    
    // 动态创建对象（使用指针）
    Rectangle* pRect = new Rectangle(6.0, 4.0);
    cout << "矩形3的面积: " << pRect->getArea() << endl;
    delete pRect;  // 释放内存
    
    return 0;
}

对象的创建方式

  • 栈上创建：ClassName obj;
  • 带参数创建：ClassName obj(param1, param2);
  • 列表初始化：ClassName obj{param1, param2}; (C++11)
  • 指针创建：ClassName* ptr = new ClassName();
  • 对象数组：ClassName objArray[5];

成员访问

  • 对象访问：obj.member
  • 指针访问：ptr->member 或 (*ptr).member
  • 常对象：只能调用常量成员函数 const ClassName obj;
  • 静态成员：不需要对象，通过类名访问 ClassName::staticMember

构造函数与析构函数

构造函数和析构函数是类的特殊成员函数，用于对象的初始化和清理。

构造函数

构造函数在对象创建时自动调用，用于初始化对象的数据成员。

class Person {
private:
    string name;
    int age;
    
public:
    // 默认构造函数
    Person() : name(""), age(0) {
        cout << "默认构造函数被调用" << endl;
    }
    
    // 带参数的构造函数
    Person(string n, int a) : name(n), age(a) {
        cout << "带参数构造函数被调用" << endl;
    }
    
    // 拷贝构造函数
    Person(const Person& other) : name(other.name), age(other.age) {
        cout << "拷贝构造函数被调用" << endl;
    }
};

析构函数

析构函数在对象被销毁时自动调用，用于释放资源或执行清理操作。

class Resource {
private:
    int* data;
    
public:
    // 构造函数
    Resource(int size) {
        data = new int[size];
        cout << "资源已分配" << endl;
    }
    
    // 析构函数
    ~Resource() {
        delete[] data;
        cout << "资源已释放" << endl;
    }
};

构造函数注意事项

  • 构造函数名必须与类名相同
  • 构造函数没有返回类型，甚至不是void
  • 可以重载构造函数，提供不同的初始化方式
  • 如果没有定义任何构造函数，编译器会提供默认构造函数
  • 使用成员初始化列表通常比在函数体内赋值更高效

访问修饰符

C++ 提供了三种访问修饰符，用于控制类成员的可见性和访问权限。

class AccessExample {
private:
    int privateVar;   // 只能被类内部访问
    
protected:
    int protectedVar; // 可被类内部和派生类访问
    
public:
    int publicVar;    // 可被任何地方访问
    
    void setPrivateVar(int val) {
        privateVar = val;  // 可以访问私有成员
    }
};

访问修饰符总结

修饰符 类内部 派生类 外部
private ✓ ✗ ✗
protected ✓ ✓ ✗
public ✓ ✓ ✓

继承

继承是面向对象编程的核心概念之一，它允许创建一个从已有类（基类）派生的新类（派生类），派生类继承基类的成员，并可以添加新的成员或修改继承的行为。

// 基类
class Shape {
protected:
    int x, y;  // 位置
    
public:
    Shape(int x = 0, int y = 0) : x(x), y(y) {}
    
    void move(int newX, int newY) {
        x = newX;
        y = newY;
    }
    
    virtual double area() const {
        return 0.0;  // 基类的默认实现
    }
    
    virtual void draw() const {
        cout << "绘制形状" << endl;
    }
};

// 派生类
class Circle : public Shape {
private:
    double radius;
    
public:
    Circle(int x, int y, double r) : Shape(x, y), radius(r) {}
    
    // 重写基类的函数
    double area() const override {
        return 3.14159 * radius * radius;
    }
    
    void draw() const override {
        cout << "绘制圆: 圆心(" << x << "," << y << "), 半径 " << radius << endl;
    }
};

继承的类型

  • 公有继承（public）：基类的公有成员在派生类中仍为公有，保护成员在派生类中仍为保护
  • 保护继承（protected）：基类的公有和保护成员在派生类中均为保护成员
  • 私有继承（private）：基类的公有和保护成员在派生类中均为私有成员

继承的应用

继承可以用于创建类的层次结构，表示"是一个"的关系，例如：

  • 动物 → 哺乳动物 → 犬科 → 狗
  • 形状 → 二维形状 → 多边形 → 矩形 → 正方形
  • 账户 → 银行账户 → 储蓄账户/支票账户

通过继承可以实现代码重用、扩展现有类和多态性。
