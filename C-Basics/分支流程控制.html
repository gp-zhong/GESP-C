C++ 分支流程控制

掌握 C++ 中的条件语句，实现程序的逻辑控制和决策

if 语句

if 语句是最基本的分支结构，它根据条件表达式的值决定是否执行特定代码块。

简单的 if 语句

// 简单的if语句
if (条件表达式) {
    // 当条件为真时执行的代码
}

// 示例：检查数字是否为正数
int number = 10;
if (number > 0) {
    cout << "这是一个正数" << endl;
}

if-else 语句

// if-else语句
if (条件表达式) {
    // 当条件为真时执行的代码
} else {
    // 当条件为假时执行的代码
}

// 示例：判断奇偶数
int num = 7;
if (num % 2 == 0) {
    cout << num << " 是偶数" << endl;
} else {
    cout << num << " 是奇数" << endl;
}

if-else if-else 语句

// if-else if-else语句
if (条件表达式1) {
    // 当条件1为真时执行的代码
} else if (条件表达式2) {
    // 当条件1为假且条件2为真时执行的代码
} else {
    // 当所有条件都为假时执行的代码
}

// 示例：根据分数评级
int score = 85;
if (score >= 90) {
    cout << "优秀" << endl;
} else if (score >= 80) {
    cout << "良好" << endl;
} else if (score >= 60) {
    cout << "及格" << endl;
} else {
    cout << "不及格" << endl;
}

if 语句的嵌套

if 语句可以嵌套使用，即在一个 if 语句的代码块中包含另一个 if 语句。

// 嵌套的if语句
int age = 25;
double income = 50000;

if (age >= 18) {
    cout << "您是成年人" << endl;
    
    if (income > 30000) {
        cout << "您需要缴纳个人所得税" << endl;
    } else {
        cout << "您无需缴纳个人所得税" << endl;
    }
}

switch 语句

switch 语句用于根据一个整型表达式的值，从多个可能的代码分支中选择一个执行。它是多重 if-else 语句的另一种形式，在某些情况下代码更加清晰。

// switch语句的基本语法
switch (表达式) {
    case 常量1:
        // 代码块1
        break;
    case 常量2:
        // 代码块2
        break;
    ...
    default:
        // 默认代码块
        break;
}

// 示例：根据月份显示天数
int month = 2;
int year = 2024;
int days;

switch (month) {
    case 1: case 3: case 5: case 7: case 8: case 10: case 12:
        days = 31;
        break;
    case 4: case 6: case 9: case 11:
        days = 30;
        break;
    case 2:
        // 检查是否是闰年
        if ((year % 4 == 0 && year % 100 != 0) || (year % 400 == 0))
            days = 29;
        else
            days = 28;
        break;
    default:
        cout << "无效的月份！" << endl;
        days = 0;
}

if (days != 0) {
    cout << year << "年" << month << "月有" << days << "天" << endl;
}

注意事项

  • switch 表达式必须是整型或枚举类型
  • case 标签必须是常量表达式
  • 如果忘记在 case 语句末尾添加 break，会导致程序继续执行下一个 case 中的代码，这被称为"穿透"（fall through）
  • default 分支是可选的，但建议总是包含 default 分支以处理未预期的情况

故意不使用 break 的情况

有时我们会故意不使用 break 来利用 switch 语句的"穿透"特性：

// 使用switch语句的穿透特性
int day = 3; // 星期三
switch (day) {
    case 1: case 2: case 3: case 4: case 5:
        cout << "工作日" << endl;
        break;
    case 6: case 7:
        cout << "周末" << endl;
        break;
    default:
        cout << "无效日期" << endl;
}

条件运算符（三元运算符）

条件运算符（也称为三元运算符）提供了一种简洁的方式来表达条件判断和赋值，是 if-else 语句的简写形式。

// 条件运算符的语法
条件表达式 ? 表达式1 : 表达式2;

// 等价的if-else语句
if (条件表达式) {
    表达式1;
} else {
    表达式2;
}

// 示例：找出两个数中的较大值
int a = 5, b = 7;
int max = (a > b) ? a : b;
cout << "较大值是: " << max << endl;

// 示例：判断考试是否通过
int score = 75;
string result = (score >= 60) ? "通过" : "不通过";
cout << "考试结果: " << result << endl;

使用指南

条件运算符在简单条件判断和赋值时非常有用，使代码更加简洁。但对于复杂的条件逻辑，使用标准的 if-else 语句通常更易读。

嵌套的条件运算符

条件运算符可以嵌套使用，但可能会降低代码的可读性：

// 嵌套的条件运算符
int a = 5, b = 7, c = 3;
int max = (a > b) ? ((a > c) ? a : c) : ((b > c) ? b : c);
cout << "三个数中的最大值: " << max << endl;

// 等价的if-else语句，更易读
int max;
if (a > b) {
    if (a > c) {
        max = a;
    } else {
        max = c;
    }
} else {
    if (b > c) {
        max = b;
    } else {
        max = c;
    }
}

短路求值

C++ 中的逻辑运算符（&& 和 ||）使用短路求值，这是一种优化机制，它可以在某些情况下避免不必要的计算。

逻辑与 (&&)

在逻辑与运算中，如果第一个操作数为假，则不会计算第二个操作数，因为整个表达式肯定为假。

// 短路求值与&&
int x = 5;
if (x > 10 && x++ > 0) {
    // 这里不会执行
}
cout << "x = " << x << endl; // 输出 x = 5，因为 x++ 没有被执行

逻辑或 (||)

在逻辑或运算中，如果第一个操作数为真，则不会计算第二个操作数，因为整个表达式肯定为真。

// 短路求值与||
int y = 5;
if (y > 0 || y++ > 10) {
    // 这里会执行
}
cout << "y = " << y << endl; // 输出 y = 5，因为 y++ 没有被执行

应用示例

短路求值在编程中非常有用，特别是在避免潜在错误的情况下： 

// 避免除以零错误
int denominator = 0;
if (denominator != 0 && (10 / denominator) > 5) {
    // 安全的执行，不会发生除以零的错误
}

// 避免空指针访问
int* ptr = nullptr;
if (ptr != nullptr && *ptr > 10) {
    // 安全的执行，不会发生空指针解引用的错误
}
