C++ 字符串

掌握字符串的概念与操作，处理文本信息的核心技能

字符串基本概念

字符串是一种重要的数据类型，用于存储和处理文本信息。在C++中，有多种处理字符串的方式。

字符串定义

字符串是将字符顺次排列形成的序列。在C++中，字符串可以通过以下方式表示：

  • C风格字符串 - 字符数组，以空字符('\0')结尾
  • C++风格字符串 - 使用标准库中的string类
// C风格字符串
char str1[] = "Hello";  // 编译器自动添加'\0'
char str2[6] = {'H', 'e', 'l', 'l', 'o', '\0'};

// C++风格字符串
#include <string>
std::string str3 = "Hello";
std::string str4("World");

重要术语

  • 子串 - 字符串的连续部分
  • 子序列 - 从字符串中提取的字符，保持相对顺序
  • 前缀 - 从串首开始到某个位置的子串
  • 后缀 - 从某个位置开始到串尾的子串
  • 字典序 - 字符串按字符逐位比较的顺序
  • 回文串 - 正读和反读都相同的字符串
  • 汉明距离 - 两个等长字符串中对应位置不同字符的数量

字符集

字符集是一个建立了全序关系的集合，其中的元素称为字符。常见的字符集包括：

  • ASCII - 128个字符，包括英文字母、数字和特殊符号
  • Unicode - 包含世界上大多数书写系统的字符
  • UTF-8 - Unicode的一种可变长度编码方式

C风格字符串

C风格字符串本质上是字符数组，以空字符('\0')结尾。C++支持C风格字符串操作，主要通过cstring（或string.h）头文件提供的函数。

基本操作 

#include <cstring>
#include <iostream>
using namespace std;

int main() {
    char str1[20] = "Hello";
    char str2[20] = "World";
    char str3[40];
    
    // 字符串长度
    cout << "str1长度: " << strlen(str1) << endl;  // 5
    
    // 字符串复制
    strcpy(str3, str1);
    cout << "str3: " << str3 << endl;  // Hello
    
    // 字符串连接
    strcat(str3, str2);
    cout << "连接后: " << str3 << endl;  // HelloWorld
    
    // 字符串比较
    int result = strcmp(str1, str2);
    if (result < 0)
        cout << "str1 < str2" << endl;
    else if (result > 0)
        cout << "str1 > str2" << endl;
    else
        cout << "str1 = str2" << endl;
    
    return 0;
}

C风格字符串的注意事项

使用C风格字符串时需要注意以下几点：
- 确保字符数组有足够空间存储字符串（包括结尾的'\0'）
- 避免内存溢出，使用安全版本的函数如strncpy, strncat
- 字符串操作函数可能导致潜在的安全问题和缓冲区溢出
- 现代C++编程中，建议优先使用std::string类

字符串与数值转换

#include <cstdlib>  // 包含数值转换函数

int main() {
    // 字符串转整数
    char num1[] = "12345";
    int value1 = atoi(num1);  // 12345
    
    // 字符串转浮点数
    char num2[] = "123.45";
    double value2 = atof(num2);  // 123.45
    
    // 字符串转长整型
    char num3[] = "123456789";
    long value3 = atol(num3);  // 123456789
    
    // 整数转字符串
    char buffer[20];
    int num = 12345;
    sprintf(buffer, "%d", num);  // 将数字转为字符串
    
    // 更安全的方式 (C++11)
    snprintf(buffer, sizeof(buffer), "%d", num);
    
    return 0;
}

C++ string 类

C++标准库提供了string类，它提供了更安全、更灵活的字符串操作，比C风格字符串更易于使用。

创建与基本操作 

#include <string>
#include <iostream>
using namespace std;

int main() {
    // 创建string对象
    string s1;                    // 空字符串
    string s2 = "Hello";          // 从字符串字面量初始化
    string s3("World");           // 使用构造函数
    string s4(5, 'A');            // 5个'A'字符: "AAAAA"
    string s5(s2);                // 拷贝构造: "Hello"
    
    // 基本操作
    cout << "s2长度: " << s2.length() << endl;       // 5
    cout << "s2大小: " << s2.size() << endl;         // 5
    cout << "s4容量: " << s4.capacity() << endl;     // >=5
    cout << "s2是否为空: " << s2.empty() << endl;    // 0 (false)
    
    // 访问字符
    cout << "s2的第一个字符: " << s2[0] << endl;     // H
    cout << "s2的最后一个字符: " << s2.back() << endl; // o
    
    // 修改string
    s1 = "Hi";                   // 赋值操作
    s2 += " C++";                // 追加文本: "Hello C++"
    s3.append(" Programming");   // 追加: "World Programming"
    
    // 遍历string的两种方式
    cout << "使用索引遍历: ";
    for (size_t i = 0; i < s2.length(); i++) {
        cout << s2[i];
    }
    cout << endl;
    
    cout << "使用迭代器遍历: ";
    for (auto c : s2) {   // C++11 范围for循环
        cout << c;
    }
    cout << endl;
    
    return 0;
}

字符串处理实例 

#include <string>
#include <iostream>
#include <algorithm>
using namespace std;

int main() {
    // 统计单词数量
    string text = "This is a sample text with some words";
    int wordCount = 1;  // 至少有一个单词
    
    for (char c : text) {
        if (c == ' ') {
            wordCount++;
        }
    }
    cout << "单词数量: " << wordCount << endl;
    
    // 反转字符串
    string str = "Hello";
    reverse(str.begin(), str.end());
    cout << "反转后: " << str << endl;  // olleH
    
    // 大小写转换
    string s = "Hello World";
    
    // 转大写
    transform(s.begin(), s.end(), s.begin(), ::toupper);
    cout << "大写: " << s << endl;  // HELLO WORLD
    
    // 转小写
    transform(s.begin(), s.end(), s.begin(), ::tolower);
    cout << "小写: " << s << endl;  // hello world
    
    // 删除空格
    string whitespace = " Hello World ";
    whitespace.erase(0, whitespace.find_first_not_of(" \t\n\r\f\v"));
    whitespace.erase(whitespace.find_last_not_of(" \t\n\r\f\v") + 1);
    cout << "去除前后空格: [" << whitespace << "]" << endl;
    
    return 0;
}

字符串算法 

// 回文检测
bool isPalindrome(const string& s) {
    int left = 0;
    int right = s.length() - 1;
    
    while (left < right) {
        if (s[left] != s[right])
            return false;
        left++;
        right--;
    }
    return true;
}

// 最长公共前缀
string longestCommonPrefix(vector& strs) {
    if (strs.empty()) return "";
    
    string prefix = strs[0];
    for (size_t i = 1; i < strs.size(); i++) {
        // 逐个比较前缀
        while (strs[i].find(prefix) != 0) {
            prefix = prefix.substr(0, prefix.length() - 1);
            if (prefix.empty()) return "";
        }
    }
    return prefix;
}

// 计算子串出现次数
int countSubstring(const string& str, const string& sub) {
    if (sub.empty()) return 0;
    
    int count = 0;
    size_t pos = 0;
    while ((pos = str.find(sub, pos)) != string::npos) {
        count++;
        pos += sub.length();
    }
    return count;
}

字符串相关的数据结构与高级应用

高级字符串应用

在算法和实际应用中，字符串处理涉及许多高级技术和数据结构：

  • 字符串匹配算法 - KMP、Boyer-Moore、Rabin-Karp等
  • 字典树(Trie) - 高效存储和检索字符串
  • 后缀数组和后缀树 - 解决复杂的字符串问题
  • 正则表达式 - 强大的字符串模式匹配工具
  • 字符串哈希 - 快速比较和查找
  • 编辑距离 - 测量两个字符串的相似度

这些技术在文本处理、搜索引擎、DNA序列分析等领域有广泛应用。

常见问题与注意事项

常见错误

  • 缓冲区溢出 - 在C风格字符串中写入超过分配空间的数据
  • 字符串终止符 - 忘记考虑'\0'的空间或意外覆盖它
  • 越界访问 - 访问string对象或字符数组的无效索引
  • 内存泄漏 - 动态分配字符串后没有释放内存
  • 字符编码问题 - 不同编码之间转换错误或不处理多字节字符

最佳实践

  • 优先使用std::string而不是C风格字符串
  • 使用reserve()预分配空间以提高性能
  • 对于大型字符串操作，考虑使用StringStream
  • 使用const string&作为函数参数避免不必要的复制
  • 处理用户输入时注意验证和净化字符串
  • 熟悉标准库中的字符串算法以提高效率
  • 处理国际化应用时注意Unicode和多字节字符集
