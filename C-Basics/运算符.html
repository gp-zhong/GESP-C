C++ 运算符

理解 C++ 中各种运算符的用法、优先级和关联性

算术运算符

算术运算符用于执行基本的数学运算，如加法、减法、乘法和除法。

运算符 描述 示例
+ 加法 a + b
- 减法 a - b
* 乘法 a * b
/ 除法 a / b
% 取模（余数） a % b
++ 自增 ++a 或 a++
-- 自减 --a 或 a-- 
// 算术运算符示例
int a = 10, b = 3;

cout << "a + b = " << a + b << endl;  // 输出: 13
cout << "a - b = " << a - b << endl;  // 输出: 7
cout << "a * b = " << a * b << endl;  // 输出: 30
cout << "a / b = " << a / b << endl;  // 输出: 3 (整数除法)
cout << "a % b = " << a % b << endl;  // 输出: 1 (余数)

float c = 10.0, d = 3.0;
cout << "c / d = " << c / d << endl;  // 输出: 3.33333 (浮点数除法)

int e = 5;
cout << "++e = " << ++e << endl;      // 输出: 6 (前缀自增)
cout << "e++ = " << e++ << endl;      // 输出: 6 (后缀自增)
cout << "e = " << e << endl;          // 输出: 7

关系运算符

关系运算符用于比较两个值，返回结果为布尔值（true 或 false）。

运算符 描述 示例
== 等于 a == b
!= 不等于 a != b
> 大于 a > b
< 小于 a < b
>= 大于等于 a >= b
<= 小于等于 a <= b 
// 关系运算符示例
int a = 10, b = 5;

cout << "a == b 的结果是 " << (a == b) << endl;  // 输出: 0 (false)
cout << "a != b 的结果是 " << (a != b) << endl;  // 输出: 1 (true)
cout << "a > b 的结果是 " << (a > b) << endl;    // 输出: 1 (true)
cout << "a < b 的结果是 " << (a < b) << endl;    // 输出: 0 (false)
cout << "a >= b 的结果是 " << (a >= b) << endl;  // 输出: 1 (true)
cout << "a <= b 的结果是 " << (a <= b) << endl;  // 输出: 0 (false)

逻辑运算符

逻辑运算符用于组合多个条件表达式，并返回布尔结果。

运算符 描述 示例
&& 逻辑与 a && b
|| 逻辑或 a || b
! 逻辑非 !a 
// 逻辑运算符示例
bool a = true, b = false;

cout << "a && b 的结果是 " << (a && b) << endl;  // 输出: 0 (false)
cout << "a || b 的结果是 " << (a || b) << endl;  // 输出: 1 (true)
cout << "!a 的结果是 " << (!a) << endl;         // 输出: 0 (false)
cout << "!b 的结果是 " << (!b) << endl;         // 输出: 1 (true)

// 短路求值
int x = 5, y = 0;
if (y != 0 && x / y > 2) {  // 短路求值，避免除以零
    cout << "x / y > 2" << endl;
}

短路求值

C++ 中的逻辑运算符具有短路求值的特性：

  • &&：如果第一个操作数为 false，则不会计算第二个操作数
  • ||：如果第一个操作数为 true，则不会计算第二个操作数

位运算符

位运算符用于对二进制位进行操作，常用于底层编程、优化和位掩码操作。

运算符 描述 示例
& 按位与 a & b
| 按位或 a | b
^ 按位异或 a ^ b
~ 按位取反 ~a
<< 左移 a << b
>> 右移 a >> b 
// 位运算符示例
int a = 5;  // 二进制 0101
int b = 3;  // 二进制 0011

cout << "a & b = " << (a & b) << endl;   // 输出: 1 (二进制 0001)
cout << "a | b = " << (a | b) << endl;   // 输出: 7 (二进制 0111)
cout << "a ^ b = " << (a ^ b) << endl;   // 输出: 6 (二进制 0110)
cout << "~a = " << (~a) << endl;         // 输出: -6 (取决于整数表示)
cout << "a << 1 = " << (a << 1) << endl; // 输出: 10 (二进制 1010)
cout << "a >> 1 = " << (a >> 1) << endl; // 输出: 2 (二进制 0010)
